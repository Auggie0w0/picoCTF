# picoCTF Practice Repository

This repository contains my solutions, notes, and tools used for completing picoCTF challenges.

## What is picoCTF?
picoCTF is a free computer security education program with original content built on a capture-the-flag framework. It provides a series of web-based challenges that help participants learn various aspects of cybersecurity and programming.

## Repository Structure
- Solutions: Detailed walkthrough of how I solved each challenge
- Notes: Important concepts and learning points
- Tools: List of tools and scripts used for solving challenges
